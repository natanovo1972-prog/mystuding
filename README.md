## Учебный проект "Виджет банковских операций"

## Описание
Проект разрабатывает новые функции для личного кабинета клиента банка. Виджет показывает последние успешные операции клиента банка.
На данном этапе в проекте реализованы следущие функции:
1. get_mask_card_number - функция принимает в качестве аргумента номер карты клиента и возвращает его маску - последние 4 цифры карты.
2. get_mask_account - функция принимает номер счета клиента и возвращает маску счета - последние 4 цифры.
3. mask_account_card - функция принимает тип карты и номер карты или номер счета и возвращает замаскированные данные.
4. get_date - функция принимает данные даты и времени и возвращает только дату.
5. filter_by_state - функция фильтрует операции по статусу выполнения.
6. sort_by_date - функция фильтрует операции по дате выполнения.
